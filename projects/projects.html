Kaggle Challenge - Toxic Comment Classification


Classified comments into toxic, severe toxic, obscene, threat, insult and indentity hate using Machine Learning and Big Data technologies. Used Apache Spark and SparkR to create a SVM model that predicts in what category/categories each comment classifies into. Model had an accuracy of 87%.
